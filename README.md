# userList

A simple web-based full-stack app:
- Get list from the backend (GET)
- Add new user (POST)
- delete the whole user list (DELETE)

Tech Stack:
- frontend: HTML, CSS, JavaScript
- backend: express

## Before Started

```shell
npm -init
npm i cors express body-parser
npm i nodemon -D
```
## Run the backend 

```shell
npx nodemon index.js
```
